The Organic Route 🌿

Empowering Your Choices for a Greener, Healthier Future
The Organic Route is a web-based platform designed to educate users about harmful chemicals in daily-use products and promote organic, chemical-free alternatives.

🛠 Tech Stack
Frontend: HTML, CSS, Tailwind CSS
Future Enhancements: JavaScript, Backend Integration

📌 Features
✅ Body Care, Healthy Food, and Medicine Alternatives – Provides safe, organic options.
✅ Harmful Chemical Awareness – Educates users on toxic ingredients to avoid.
✅ User-Friendly UI – Clean and structured interface for better readability.
